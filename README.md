# AEVO AdA Lern-App

Statische Browser-App im Führerschein-Lernstil für die IHK-Ausbildereignungsprüfung / AEVO.

## Funktionen
- Fragen nach Handlungsfeld filtern
- Lernmodus mit sofortiger Erklärung
- Prüfungsmodus mit Endauswertung
- Fehlerliste im Browser-Speicher
- GitHub-Pages-fähig ohne Backend

## Lokal starten
Öffne `index.html` direkt im Browser.

## Auf GitHub Pages veröffentlichen
1. Neues GitHub Repository erstellen, z. B. `aevo-lernapp`.
2. Diese Dateien hochladen: `index.html`, `styles.css`, `app.js`, `questions.js`, `README.md`.
3. In GitHub: Settings → Pages → Deploy from branch → `main` und `/root` wählen.
4. Nach kurzer Zeit ist die App über die GitHub-Pages-URL erreichbar.

## Hinweis
Die Fragen sind eigene Lernfragen auf Basis der AEVO-Handlungsfelder. Sie sind keine offiziellen IHK-Prüfungsfragen und enthalten keine kopierten Buchinhalte.
