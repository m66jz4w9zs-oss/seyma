# AEVO MasterKit – AdA Lernapp

Professionelle statische Lernapp für die IHK-Ausbildereignungsprüfung / AEVO.

## Funktionen
- 368 eigenständige AEVO-Lernfragen nach HF1–HF4
- Lernmodus mit Sofort-Erklärung
- Prüfungsmodus mit 80 zufälligen Fragen
- Score je Prüfung
- Auswertung nach Handlungsfeldern
- Fehlertraining
- Lokale Speicherung im Browser per localStorage
- Glückliche Katze bei richtig, traurige Katze bei falsch
- GitHub-Pages-fähig

## Nutzung lokal
`index.html` im Browser öffnen.

## GitHub Pages
1. Neues GitHub Repository erstellen
2. Alle Dateien aus diesem Ordner hochladen
3. Settings → Pages → Deploy from branch → main / root
4. Die angezeigte GitHub-Pages-URL öffnen

## Hinweis
Die Fragen sind eigenständig erstellt und orientieren sich an AEVO-Handlungsfeldern und typischen IHK-Prüfungsthemen. Es werden keine Buchtexte oder offiziellen IHK-Prüfungsfragen kopiert.
