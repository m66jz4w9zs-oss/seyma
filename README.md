# AEVO MasterKit Pro

Responsive Lern-App für die Ausbildereignungsprüfung / AdA.

## Inhalte
- Über 370 Multiple-Choice-Fragen
- Über 120 Situationsaufgaben, davon offene Fälle und Kombifälle
- Prüfungsmodus: 80 MC-Fragen + 3 Situationsfälle
- Score je Prüfung und Prüfungsverlauf
- Auswertung nach HF1-HF4
- Fehlertraining
- lokale Speicherung im Browser
- responsive UI für Desktop und Smartphone
- glückliche und traurige Katzen als Feedback

## GitHub Pages
1. Dateien in ein Repository hochladen.
2. `index.html`, `styles.css`, `app.js`, `questions.js` müssen im Root liegen.
3. Settings → Pages → Deploy from branch → main / root.
4. Nach dem Build ist die App unter der GitHub-Pages-URL erreichbar.

## Hinweis
Die Fragen sind eigenständig formuliert und orientieren sich an den AEVO-Handlungsfeldern. Es wird kein urheberrechtlich geschützter Buchtext kopiert.
